# DSA & Coding Patterns — Interview Prep

> The #1 filter at product-based companies (Amazon, Google, Flipkart, Razorpay, Swiggy, etc.)

## Must-Know Patterns (ordered by frequency)

1. **Arrays & Hashing** — Two Sum, Kadane's, Sliding Window
2. **Two Pointers** — Container with most water, 3Sum
3. **Sliding Window** — Max subarray, minimum window substring
4. **Stack & Queue** — Valid parentheses, next greater element
5. **Binary Search** — Rotated sorted array, search in matrix
6. **Linked List** — Reverse, detect cycle, merge sorted
7. **Trees & BST** — Traversals, LCA, validate BST
8. **Graphs** — BFS/DFS, topological sort, shortest path
9. **Dynamic Programming** — Fibonacci, knapsack, LCS, coin change
10. **Backtracking** — N-Queens, permutations, subsets
11. **Heap/Priority Queue** — Top-K, median of stream
12. **Greedy** — Activity selection, Huffman coding

## Files

- `qa.md` — Pattern explanations + 50 curated problems (Easy→Hard)
- `solutions/` — Your implementations

## Study Strategy

- Solve 2-3 problems/day minimum
- Focus on patterns, not memorizing solutions
- Time yourself: Easy ≤15min, Medium ≤25min, Hard ≤45min
- Always write brute force first, then optimize
